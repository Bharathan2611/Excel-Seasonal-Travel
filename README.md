# Excel Seasonal Travel Data Analysis

## Overview
This project analyzes travel booking and customer data using Microsoft Excel.  
The goal is to extract insights on travel trends, customer behavior, booking sources, costs, discounts, ratings, and seasonal preferences.

This project demonstrates practical Excel skills, including data cleaning, formulas, Pivot Tables, and visualizations.

## Tools Used
- Microsoft Excel

## Dataset Description
The dataset contains travel records with fields such as:
- Travel ID, Date, Season
- Customer details (ID, Name, Age, Gender)
- Travel details (destination, region, country, city)
- Trip specifics (duration, group size, travel mode, accommodation, meal plan)
- Financials (trip cost, discount)
- Ratings and feedback
- Booking source and status
- Referral and special requests

## Excel Skills Demonstrated
### Data Preparation
- Remove blanks and errors
- Standardize date formats
- Fill or flag missing values

### Formulas and Functions
- SUM, AVERAGE, COUNT
- IF, COUNTIF, SUMIF
- TEXT functions
- VLOOKUP or XLOOKUP

### Pivot Tables and Analysis
- Count of trips by destination and season
- Average trip cost by region
- Booking status by source
- Rating summaries by travel agent

### Charts and Visualization
- Bar charts for popular destinations
- Line charts for seasonal trends
- Pie charts for booking source share

## Insights
Using the dataset, you can:
- Identify most preferred travel locations
- Compare cost patterns across seasons
- Find trends in bookings (online vs agency vs walk-in)
- Correlate discounts with customer ratings

## How to Use
1. Download the Excel file from this repository
2. Open using Microsoft Excel
3. Look at the cleaned dataset sheet
4. Review Pivot Tables and Charts
5. Explore insights and interpretations
